# Documentação do Site de Casamento (Data-Driven & PT-BR)

## Introdução
Este projeto é um site de casamento moderno, **totalmente dinâmico** e com **código 100% em Português**. Foi otimizado para performance, facilidade de manutenção e responsividade extrema em dispositivos móveis.

---

## Arquitetura de Dados (✅ Versão Brasileira v3.2)
Diferente de templates comuns, este projeto foi totalmente traduzido internamente para facilitar o entendimento de desenvolvedores e usuários brasileiros:
- **Arquivo de Configuração:** `assets/data/content.json`
- **Chaves em Português:** Todas as chaves do JSON (ex: `inicio`, `galeria`, `traje`, `confirmacao`) estão em PT-BR.
- **Código Limpo:** Variáveis JavaScript, IDs de HTML e Classes CSS seguem o padrão da língua portuguesa (ex: `preencherConteudoSite`, `barra-navegacao`, `grade-presentes`).

---

## Funcionalidades e Design

### 1. Navegação Horizontal (Mobile)
- Barra de navegação compacta de **40px**.
- Itens de menu sequenciais e horizontais, otimizados para não cortarem em telas estreitas.

### 2. Monetização & AdSense
- Estrutura pronta para anúncios do Google.
- Arquivo `robots.txt` incluso na raiz.
- Seções de conteúdo expandidas para cumprir as políticas de "Conteúdo de Valor" do AdSense.

### 3. Gestão Centralizada
- Alteração de fotos, nomes, datas e links de afiliados feita exclusivamente através de um único arquivo JSON.

---

## Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3, JavaScript Vanilla.
- **Dados:** JSON (Estrutura em PT-BR).
- **Status Final:** 100% Traduzido, Responsivo e Otimizado (v3.2).
